# KeNaWPWALanding Page

This repository contains the source code for the **KeNaWPWA Landing Page**, built with **React**. The landing page serves as a gateway to the **KeNaWPWA** platform, providing information about the association's mission, initiatives, collaboration with CEJAD, and more. It is designed to promote awareness and engagement for waste pickers and their communities.

### Features:
- **Logo**: Represents the KeNaWPWA branding.
- **About KeNaWPWA**: Brief introduction to the association's goals and mission.
- **Collaboration with CEJAD**: Highlights the partnership with the Center for Environmental Justice and Development (CEJAD).
- **Initiatives**:
  - **Member Registration**: Section for new members to sign up.
  - **Environmental Awareness**: Information on environmental issues, including carbon footprints and global warming.
  - **Policy Advocacy**: Promoting better waste management systems and advocating for waste pickers' rights.
  - **Community Clean-Up Drives**: Encouraging community involvement through regular clean-up events.
  - **Training Programs**: Training opportunities in Kisumu, Nairobi, Nakuru, with plans for expansion.
- **App Download Section**: QR codes for downloading the **Waste Pickers App** and **Waste Pickersâ€™ Managers App**.
- **Dashboard/Web Login**: Links to the main platform for admins, managers, and waste pickers.

### Tech Stack:
- **React**: Frontend framework used for building the landing page.
- **CSS/SCSS**: Styling and layout.
- **SVGs**: Icons used throughout the page.
- **QR Codes**: For app download links.
